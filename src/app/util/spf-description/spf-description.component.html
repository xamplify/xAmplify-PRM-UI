Godaddy SPF Configuration :

We have API-based settings for adding or updating SPF entries in GoDaddy.


Manual SPF Configuration :

Add an SPF Record:

v=spf1 include:u10208008.wl009.sendgrid.net ~all
v=spf1 include:_spf.google.com include:u10208008.wl009.sendgrid.net ~all
v=spf1 include:spf.protection.outlook.com include:u10208008.wl009.sendgrid.net ~all
